# Audio Previewer TUI

Quickly preview your audio files!

## Installation

`cargo install audio-preview`

## Usage

`listen` for tui

or

`listen <path>` to quickly play a file

## Keybinds

| Keybind                               | Usage                        |
|---------------------------------------|------------------------------|
| Q                                     | Quit                         |
| Arrow Keys                            | Navigate                     |
| Tab                                   | Switch directories and files |
| L                                     | Toggle looping               |
| Right Arrow Key or Enter in file mode | Play file                    |
